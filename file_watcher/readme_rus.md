
Программа которая обнаруживает изменения в дирикториях и/или файлах.
Если произошло изменение то однократно запускает программу.
Использует системное слежение за файлами (модуль python watchdog).


Файл конфигурации
======================

Конфигурация состоит из 3 обязательных секций: commands, files, dirs.

Секция commands
------------------
Содержит список команд, каждая команда на отдельной строке, имена команд должны отличаться.
В формате:
    ИмяКоманды: 'Текст команды'

Секция files
-------------------
Файлы для отслеживания.

В формате:
    'ПутьФайла': ИмяКоманды
Или
    'ПутьФайла': [ИмяКоманды, ИмяКоманды]

Секция dirs
---------------
Дириктории для отслеживания.

В формате:
    'ПутьДириктории': ИмяКоманды
Или
    'ПутьДириктории': [ИмяКоманды, ИмяКоманды]


Пример
---------

```
commands:
  cmd1: 'echo file changed'
  cmd2: 'echo dir changed'
  cmd3: 'echo file changed'
files:
  'path_to_file': cmd1
  'path_to_file2': [cmd1, cmd3]
dirs:
  'path_to_dir': cmd2
```
